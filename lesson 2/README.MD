## Нужно Написать один кастомный дашборд, назови его "My custom dash for bank monitoring"

_Писать нужно только из кастомных метрик (создать дашборд + создать виджет + простая promql формула), то есть из сырых и больше не из чего: `localhost:9090/metrics` и т.д..., то есть у тебя нет никаких данных кроме `/metrics` - ты дергаешь данные только оттуда. А также у ИИ можно спросить к какому типу метрик относится твоя метрика, когда будешь искать, чтобы легче было выбирать виджеты под дашборд. На какждый запрос ниже минимум по 2 метрики. То есть всего дашборд должен состоять из 22 метрик или 22 виджетов, как минимум, если больше - лучше, допустим будет 3-4 метрики по HTTP - можно взять все, но и не делать "пирог!"._

1. Напиши кастомный дашборд который считает (Prometheus):
    - TSDB данные Prometheus
    - Производительность HTTP-запросов
    - Системные и Go-метрики
    - Служебная информация / info
2. Считает метрики (Grafana):
    - Состояние системы и статистика
    - Работа с дашбордами и данными
    - Производительность HTTP-запросов
    - Служебная информация / info
3. Считает метрики (Node Exporter):
    - Система (System)
    - Сеть (Network)
    - Диски (Disk)
4. Дашборд должен логически разделяться на подразделы и заголовки, можно комбинировать похожие метрики между собой в один раздел, то есть полет фантазии
