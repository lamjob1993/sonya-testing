## Нужно Написать один кастомный дашборд, назови его "My custom dash for bank monitoring"

---

_Писать нужно только из кастомных метрик, то есть из сырых и больше не из чего: `localhost:9090/metrics` и т.д..., то есть у тебя нет никаких данных кроме /metrics - ты дергаешь данные только оттуда._

1. Напиши кастомный дашборд который считает (Prometheus):
    - TSDB данные Prometheus
    - Производительность HTTP-запросов
    - Системные и Go-метрики
    - Служебная информация / info
2. Считает метрики Grafana:
    - Состояние системы и статистика
    - Работа с дашбордами и данными
    - Производительность HTTP-запросов
    - Служебная информация / info
3. Считает метрики:
    - Система (System)
    - Сеть (Network)
    - Диски (Disk)
4. Дашборд должен логически разделяться на подразделы и заголовки, можно комбинировать похожие метрики между собой в один раздел, то есть полет фантазии
