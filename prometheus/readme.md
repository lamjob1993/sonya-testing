1. Скачать Prometheus последней версии и разархивировать
2. Повесить на него Unit файл на автозапуск и проверить
3. Познакомиться с командами утилиты systemctl и с запуском юнитов
4. Не забыть в юните прописать дефолтный путь к конфигу (конфигурационный файл Прометеуса), официально `/etc/prometheus/prometheus.yml` это и есть путь к конфигу, соответственно создать там папку, создать конфиг и сложить его по этому пути
5. За конфигами в ИИ
6. За помощью к написанию юнитов в ИИ
7. Знать как минимум 3-4 основные директивы юнита
